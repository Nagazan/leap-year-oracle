# Leap Year Oracle

A beginner-friendly C program that determines whether a given year is a leap year.

##  How it works
- Prompts the user to input a year.
- Uses conditional logic to check if it's a leap year.
- Displays the result.

##  Leap Year Rules:
A year is a leap year if:
- It's divisible by 4 and not divisible by 100,
- **OR** it's divisible by 400.

###  Example Output:

Enter a year: 2024
2024 is a leap year
